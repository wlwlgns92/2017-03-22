CSS : 캐스케이팅 스타일 시트[HTML 꾸미기(레이아웃.효과)]

1. css사용방법 :

  1. header 태그 안에서 style태그를 사용해 정의
  2. html 태그에서 style 태그 사용
  3. css 파일 작성해 불러오기

2. css 선택자 : 적용할 대상

  1. *{속성 = "속성값"} : 모든 곳에 효과
  2. 태그명 {속성 = "속성값"} : 모든 태그에 전역 효과
  3. .클래스명{속성 = "속성값"}: 특정 클랙스명에 효과(자주 사용되는 곳에)
  4. #id명{ 속성 = "속성값"} : 특정 id명에(특정 클래스,태그) 효과
  5. 차이 : 1. 동일한 효과가 있을경우 : 우선순위 [1.id 2.class 3.마크업[태그]]
  6. class : 자주 사용되는 스타일에 사용 / id : 한페이지에 한번만 사용가능
